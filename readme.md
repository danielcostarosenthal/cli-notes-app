# CLI Notes App

Built with Node.js.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the dependencies.

```bash
npm i
```

## Usage

```javascript
# create a note
node app.mjs add --title="Title" --body="Body"

# remove a note
node app.mjs remove --title="Title"

# read a note
node app.mjs read --title="Title"

# list out all notes
node app.mjs list


```
